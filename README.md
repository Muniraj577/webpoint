## Installation
```bash
git clone git@github.com:Muniraj577/webpoint.git

npm install
```
### Install Composer
```bash
composer install
```

### Generate Key
```bash
php artisan key:generate
```

### Migrate Database
```bash
php artisan migrate
```

### Running application
```bash
php artisan serve

npm run dev
```
